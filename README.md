# Speranza Rust Server

Next.js landing page for Speranza 10X PVE+ Rust Server.

## Local run
```bash
npm install
npm run dev
```

## Deploy
Push this project to GitHub, then Vercel will deploy automatically.

## Assets
- `public/phoenix-logo.png` is the official phoenix asset used in the hero and navbar.
- `public/speranza-logo-full.png` is a full logo lockup asset.
